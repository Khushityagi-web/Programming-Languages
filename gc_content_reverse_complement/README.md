This script computes the GC content of a DNA sequence and generates its reverse complement.
It is part of my foundational Python exercises, where I practiced applying basic string operations and dictionaries to biologically meaningful tasks.

## What the Script Does

1. GC Content Calculation

Counts the number of G and C bases and returns GC percentage.

2. Reverse Complement

Reverses the DNA sequence

Maps each base to its complement using a dictionary

Joins the result into a final string

Example:

Input:
AATCGGGACTTCAT

Output:

GC Content: 42.86%
Reverse Complement: ATGAAGTCCCGATT
