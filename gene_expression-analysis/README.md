# Gene Expression Loop Exercises in Python (Beginner Practice)

This script contains two introductory Python exercises that use gene expression values to practice looping techniques (for and while).
Although the examples use biological numbers, the primary goal is to build strong foundational programming skills for future bioinformatics workflows.
